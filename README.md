# Gym Tic Tac Toe
Classic Tic Tac Toe game environment with 3x3 board for RL against customizable opponent.

# Installation

```
git clone https://github.com/Zenchiyu/gym_tic_tac_toe.git
cd gym_tic_tac_toe
pip install -e .
```
